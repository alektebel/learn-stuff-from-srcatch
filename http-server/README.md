# HTTP Server From Scratch

This directory contains a from-scratch implementation of an HTTP server.

## Goal
Build an HTTP server to understand:
- TCP/IP networking
- HTTP protocol specification
- Request parsing and response generation
- Concurrent connection handling
- Web server architecture

## Learning Path
1. **TCP Server** - Socket programming basics
2. **HTTP/1.1 Parser** - Parse HTTP requests
3. **Response Generation** - Build HTTP responses
4. **Static File Serving** - Serve HTML, CSS, JS files
5. **Routing** - URL path handling
6. **Middleware** - Request/response pipeline
7. **Concurrency** - Handle multiple connections (threads/async)
8. **HTTPS** - TLS/SSL encryption
9. **HTTP/2** - Modern protocol features
10. **Performance** - Caching, compression, keep-alive

## Features to Implement
- GET, POST, PUT, DELETE methods
- Headers parsing
- Query parameters
- Request body parsing (form data, JSON)
- Status codes (200, 404, 500, etc.)
- Content-Type handling
- Logging and error handling
- Configuration management
